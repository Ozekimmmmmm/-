# 名古屋市におけるカフェ立地の空間分析

## ■ 概要
本プロジェクトでは、名古屋市におけるカフェの立地分布を分析し、都市構造との関係性を明らかにすることを目的とした。

GISデータとPythonを用いて、カフェの分布、密度、都心集中の傾向を定量的に評価した。

---

## ■ 背景・目的
カフェは人流や都市機能と密接に関係して立地していると考えられる。

本分析では、
- 都心部にカフェは集中しているのか
- 住宅地ではどのように分布しているのか

を明らかにすることを目的とした。

---

## ■ 使用データ
- OpenStreetMap（カフェデータ）
- GISで抽出・加工したCSVデータ（3185件 → 名古屋市内1045件に絞り込み）

---

## ■ 分析手法
1. GIS（QGIS）を用いてカフェデータを抽出し、CSV形式で出力
2. Pythonでデータを読み込み（pandas）
3. 名古屋市内のカフェにフィルタリング
4. 金山・名駅・伏見・丸の内・鶴舞を結ぶ五角形エリアを設定
5. 五角形内外のカフェ数をカウント
6. 面積を算出し、カフェ密度（件/km²）を計算

---

## ■ 結果
- 名古屋市内のカフェ数：1045件  
- 五角形内のカフェ数：156件  
- 五角形内カフェ密度：約13.8件/km²  
- 名古屋市全体の平均密度：約3.2件/km²  

 都心コアでは、市全体の約4倍の密度でカフェが分布していることが確認された。

---

## ■ 考察
- 名駅・伏見・丸の内周辺はオフィスが集積しており、人流が多いためカフェの需要が高いと考えられる。
- 金山・鶴舞などの拠点も含め、都市の主要結節点に沿ってカフェが集積している。
- 一方、住宅地では広小路通や八熊通などの幹線道路沿いに分布しており、車利用や視認性が立地に影響している可能性がある。
